# Ejercicios — Tablas de Hash

Cada ejercicio usa la interfaz `HashTable[K comparable, V any]` del paquete
[`hashtable`](https://github.com/untref-ayp2/data-structures/tree/main/hashtable).
Elegí la implementación que más sentido tenga para cada caso
(`HashTableOpenAddressing` o `HashTableChaining`).
